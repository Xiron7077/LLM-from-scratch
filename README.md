# Transformers & RAG Project

This repository contains notebooks, data, and resources for working with Transformer architectures and Retrieval-Augmented Generation (RAG).

## Project Structure

- `Transformers.ipynb`: Jupyter notebook containing experiments, explorations, or training routines for Transformer models.
- `RAG.ipynb`: Jupyter notebook demonstrating a Retrieval-Augmented Generation pipeline or related experiments.
- `scitldr_train.jsonl`: Training data in JSON lines format (commonly used for tasks such as scientific paper summarization).
- `Reports/`: Directory for storing generated reports, evaluation metrics, or outputs from the notebooks.

## Getting Started

1. Ensure you have Python installed.
2. It's recommended to use a virtual environment (`venv` or `conda`).
3. Install standard machine learning requirements for these notebooks (e.g., `torch`, `transformers`, `jupyter`, etc.) if not already installed.
4. Launch Jupyter or open the notebooks in VS Code to run the code.

```bash
# Example local setup
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt # (if a requirements file is added later)
jupyter notebook
```
